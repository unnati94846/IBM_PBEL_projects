# SMS Spam Detection

## Project Overview
This project uses Machine Learning to classify SMS messages as **Spam** or **Ham (Not Spam)**.

## Dataset
- Dataset: SMS_SPAM detection
- Features:
  - Message
  - Label (Spam/Ham)

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Workflow
1. Load Dataset
2. Data Preprocessing
3. Text Vectorization (TF-IDF/CountVectorizer)
4. Train-Test Split
5. Model Training
6. Prediction
7. Model Evaluation

## Files
- `UnnatiYadav_SMS Detection.ipynb` – Jupyter Notebook
- `SMS_SPAM detection.csv` – Dataset
